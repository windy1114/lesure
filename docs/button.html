Buttons

Simple CSS for buttons.

默认button

给html元素添加.lesure-btn即可， 但最好是给<a> 或者<div>

A Lesure Button 
<a href="#" class="lesure-btn">A Lesure Button</a>                                                             <button type="button" class="lesure-btn">A Lesure Button</button>

通过组合lesure-btn、primary-btn、large-btn、small-btn、xsmall-btn、blue-btn、orange-btn可以配置出多种大小颜色的按钮。
